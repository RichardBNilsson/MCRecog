# MCRecog
Mod that adds Speech Recognition to Minecraft
Here are the spoken commands and the corresponding key presses, for use in setting up VoiceAttack.
- Food      X
- Gone      R
- Hole      B
- Mine      Y
- Lava      U
- Night     I
- Drop      O
- Jump      V
- Heart     ,
- Jail      K
- Rot       J
- Bone      H
- Dead      G
- End       Z
- Dragon    M
- Boat      N
